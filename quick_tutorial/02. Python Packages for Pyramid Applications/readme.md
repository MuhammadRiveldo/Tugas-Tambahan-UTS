## **02: Python Packages for Pyramid Applications**
Aplikasi dipecah menjadi paket Python agar lebih terstruktur dan mudah dikembangkan. Setiap bagian aplikasi (views, routes, templates) disusun dalam direktori khusus.

**Analisis:**
Tahap ini memperkenalkan cara mengubah aplikasi single-file menjadi paket Python yang lebih terstruktur.
Pyramid menggunakan pendekatan berbasis paket (package) untuk skala besar.

Kita mempelajari:

- Struktur direktori Python (tutorial/, __init__.py, setup.py).
- File setup.py digunakan agar aplikasi bisa diinstal dengan pip install -e ..
- Keuntungan modularisasi: mudah diuji, dikonfigurasi, dan di-deploy.

Konsep ini menyiapkan dasar untuk proyek Pyramid profesional yang bisa dikembangkan, diuji, dan dipelihara dengan baik.
