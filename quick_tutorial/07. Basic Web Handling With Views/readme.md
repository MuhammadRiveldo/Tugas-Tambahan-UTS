## **07: Basic Web Handling With Views**
Memperkenalkan konsep view dan route di Pyramid. Setiap URL dipetakan ke fungsi view tertentu.

**Analisis:**
Langkah ini menjelaskan konsep views lebih dalam.
View adalah fungsi Python yang menangani URL tertentu.

Kita belajar:
- Dekorator @view_config untuk mengaitkan fungsi dengan route
- Mengembalikan dictionary ke template atau objek Response.

Intinya: view adalah jantung Pyramid yang menentukan apa yang dikirim ke pengguna.
